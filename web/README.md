# OpenmarketUI — Web

Landing page for OpenmarketUI, deployed to Cloudflare Pages via Astro.

## Commands

| Command       | Action                                      |
| :------------ | :------------------------------------------ |
| `bun install` | Install dependencies                        |
| `bun dev`     | Start local dev server at `localhost:4321`  |
| `bun build`   | Build to `./dist/`                          |
| `bun preview` | Preview build locally before deploying      |

## Deploy

Deployed automatically to Cloudflare Pages. Config in `wrangler.jsonc`.
